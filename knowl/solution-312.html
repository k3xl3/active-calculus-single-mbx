Solution 5.3.5.2.1.

\(\int\,7 x^{5} \sin(x^{6})\,dx\text{.}\) We use substitution with \(w = x^{6}\) to find this. We have \(dw = 6\,x^{5}\,dx\text{,}\) so that
\begin{equation*} \int\,7 x^{5} \sin(x^{6})\,dx = {7\over 6}\int\,\sin(w)\,dw = -{7\over 6}\,\cos(w) + C = -{7\over 6}\,\cos(x^{6}) + C. \end{equation*}
