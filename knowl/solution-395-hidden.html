\begin{equation*} \hbox{Volume of slab } = \pi r^2\Delta h = 16 \pi \Delta h \end{equation*}
\begin{equation*} \hbox{Distance to raise } = h \end{equation*}
\begin{equation*} \hbox{Work to raise } = (16 \pi)(50)(h) \Delta h. \end{equation*}
Integrating the work over all such slabs, we have
\begin{equation*} \hbox{Work} = \int_{12}^{22} (50)(16\pi)(h) \,dh = 400\pi h^2{\bigg|}_{12}^{22} = 136000 \pi \hbox{ lbf}. \end{equation*}
