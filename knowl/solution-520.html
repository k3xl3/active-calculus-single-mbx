Solution 8.5.6.1.1.

\(f(x) = \sin\!\left(4x\right)\) Then \(f(0) = \sin(0) = 0\text{,}\) and
\begin{equation*} f'(x) = 4\cos\!\left(4x\right),\qquad \hbox{so}\quad f'(0) = 4, \end{equation*}
\begin{equation*} f''(x) = -16\sin\!\left(4x\right),\qquad \hbox{so}\quad f''(0) = 0, \end{equation*}
\begin{equation*} f'''(x) = -64\cos\!\left(4x\right),\qquad \hbox{so}\quad f'''(0) = -64, \end{equation*}
\begin{equation*} f^{4}(x) = 256\sin\!\left(4x\right),\qquad \hbox{so}\quad f^{4}(0) = 0, \end{equation*}
\begin{equation*} f^{5}(x) = 1024\cos\!\left(4x\right),\qquad \hbox{so}\quad f^{5}(0) = 1024, \end{equation*}
Thus,
\begin{equation*} P_{1}(x) = 4 x, \end{equation*}
\begin{equation*} P_{3}(x) = 4 x - {32\over3} x^3, \end{equation*}
and
\begin{equation*} P_{5}(x) = 4 x - {32\over3} x^3 + {128\over15} x^5. \end{equation*}
