Activity 7.4.4.

Solve each of the following differential equations or initial value problems.

  1. \(\displaystyle \frac{dy}{dt} - (2-t) y = 2-t\)

  2. \(\displaystyle \frac{1}{t}\frac{dy}{dt} = e^{t^2-2y}\)

  3. \(y' = 2y+2\text{,}\)  \(y(0)=2\)

  4. \(y' = 2y^2\text{,}\)  \(y(-1) = 2\)

  5. \(\frac{dy}{dt} = \frac{-2ty}{t^2 + 1}\text{,}\)   \(y(0) = 4\)
