\(\displaystyle c_n = \frac{f^{(n)}(0)}{n!}\text{,}\) e.g. \(\displaystyle c_0 = \frac{f(0)}{0!} = 0\text{.}\) \(\displaystyle c_1 = \frac{f'(0)}{1!}\text{;}\) \(\displaystyle f'(x)=5 \arctan(3 x) + 5 x\frac{1}{(3 x)^2+1}3 = 5 \arctan(3 x) + \frac{15 x}{9 x^2+1}\text{;}\) \(c_1 = 0\text{.}\) \(\displaystyle c_2 = \frac{f''(0)}{2!}\text{;}\) \(\displaystyle f''(x) = \left( 5 \arctan(3 x) + \frac{15 x}{9 x^2+1} \right)'\)\(\displaystyle = \frac{15}{9 x^2+1} + \frac{15(9 x^2+1)-15 x9\cdot 2x}{(9 x^2+1)^2}\)\(\displaystyle = \frac{135 x^2 + 15}{(9 x^2+1)^2} + \frac{15 - 135 x^2}{(9 x^2+1)^2} = \frac{30}{(9 x^2+1)^2}\text{;}\) \(\displaystyle c_2 = \frac{30}{2} = 15\text{;}\) and so on. Another way is to use the power series for arctan: \(\displaystyle \arctan(t) = \sum_{n=0}^{\infty} (-1)^n \frac{t^{2n+1}}{2n+1} = t-\frac{t^3}{3}+\frac{t^5}{5}-\frac{t^7}{7}+...\text{.}\) Then \(\displaystyle 5 x \arctan(3 x) = 5 x \sum_{n=0}^\infty (-1)^n \frac{(3 x)^{2n+1}}{2n+1} = \sum_{n=0}^\infty (-1)^n \frac{5\cdot 3^{2n+1}}{2n+1} x^{2n+2}\)\(\displaystyle = 15 x^2 - \frac{135}{3} x^4 + \frac{1215}{5} x^6 - \frac{10935}{7} x^7+...\text{.}\) So \(c_0=0\text{,}\) \(c_1=0\text{,}\) \(c_2=15\text{,}\) \(c_3=0\text{,}\) \(c_4=-\frac{135}{3}=-45\text{...}\) This is an alternating series. It converges if and only if \(\displaystyle \lim_{n\to\infty}\frac{5 \cdot (3 x)^{2n+1}}{2n+1}=0\text{.}\) This limit is equal to 0 if and only if \(|3 x| \le 1\text{,}\) so \(\displaystyle |x|\le \frac{1}{3}\text{.}\) Therefore the radius of convergence is \(\displaystyle \frac{1}{3}\text{.}\)
