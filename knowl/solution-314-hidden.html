\(w = e^{5 x} + 1\text{.}\) Then \(dw = 5\,e^{5 x}\,dx\text{,}\) so that
\begin{equation*} \int\,\frac{e^{5 x}}{1 + e^{5 x}}\,dx = {1\over 5}\int\,{1\over w}\,dw = {1\over 5}\,\ln(|w|) + C = {1\over 5}\,\ln(|e^{5 x} + 1|) + C. \end{equation*}
Because \(e^{5 x} + 1\) is always positive, we can drop the absolute values from the argument of the natural log.
