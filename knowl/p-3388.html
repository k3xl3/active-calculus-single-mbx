To determine which graphed function is the the derivative of another, we use the fact that the derivative is zero where the original function has a maximum or minimum. We note that C has zeros where B has a maximum or minimum, so it could be the derivative, so we can have \(C = B'\text{.}\) In addition, B has zeros where A has a maximum or minimum, so that it is possible that \(B = A'\text{.}\)
