Exercise 3. Choosing solution of \(dy/dt=k(1-Ay)\).

Let \(A\) and \(k\) be positive constants.

Which of the given functions is a solution to \(\frac{dy}{dt}=-k(y+A)\text{?}\)

  • \(\displaystyle y = A + C e^{-kt}\)

  • \(\displaystyle y = -A + C e^{kt}\)

  • \(\displaystyle y = A^{-1} + C e^{Akt}\)
