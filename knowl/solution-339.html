Solution 5.5.5.3.1.

\begin{equation*} \frac{8 x + 6}{x^2-3x+2} = \frac{8 x+6}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}. \end{equation*}
Multiplying by \((x-1)\) and \((x-2)\text{,}\) this becomes
\begin{equation*} 8 x+6 = A(x-2)+B(x-1), \end{equation*}
or
\begin{equation*} A+B = 8 \quad\hbox{and}\quad -2A - B = 6. \end{equation*}
Solving this system yields \(A = -14\) and \(B = 22\text{.}\) Then
\begin{equation*} \int\frac{8 x+6}{x^2-3x+2}dx = \int\left(\frac{22}{x-2} - \frac{14}{x-1}\right)\,dx = 22\ln\!\left(\left|x-2\right|\right)-14\ln\!\left(\left|x-1\right|\right)+C. \end{equation*}
